## Легенда символов

```
███  - Враг (полная непрозрачность)
▓▓▓  - Враг (75% непрозрачности)
▒▒▒  - Враг (50% непрозрачности)
░░░  - Враг (25% непрозрачности)

✨   - Вспышка света
💥   - Взрыв/удар
✧    - Частица ауры
⚠    - Предупреждение (критический HP)
↑↓←→ - Направление движения
↔↕   - Движение туда-сюда
↻    - Вращение
🐉   - Босс (пример)
```
